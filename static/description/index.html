Module to set an authorization code

for using discounts from POS UI.

In order to use this module correctly, you must access to: Point of Sale application. Then open the POS configuration screen to set the authorized code, as follows:

Then you can open the related POS UI and perform the sale process. If you are willing to apply discounts for that sale, press the Discount button. You will be shown a screen to enter the authorization code.

After that, if the code is correct, you can proceed with the sale process.
